# Ícones da Extensão

Para completar a extensão, você precisa criar 3 ícones PNG:

1. **icon16.png** - 16x16 pixels (ícone pequeno na barra de ferramentas)
2. **icon48.png** - 48x48 pixels (ícone na página de extensões)
3. **icon128.png** - 128x128 pixels (ícone na Chrome Web Store)

## Dica
Você pode usar o logo da Quality Frotas ou criar um ícone simples com as iniciais "QF" em um fundo azul.

Ferramentas gratuitas para criar ícones:
- https://www.canva.com
- https://favicon.io/favicon-generator/
- https://iconifier.net/
